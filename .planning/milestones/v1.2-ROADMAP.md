# Milestone v1.2: Complete Experience

**Status:** SHIPPED 2026-01-19
**Phases:** 12-18
**Total Plans:** 10

## Overview

Transform from helper tool into full-featured, multilingual Wordle experience with multiple game modes and progressive hints. Added header navigation, settings consolidation, Open Mode with variable word lengths, progressive hints system, and Portuguese language support.

## Phases

### Phase 12: Header Layout

**Goal**: Restructure layout with focus on puzzle and keyboard as primary elements, add header nav with Stats/How-to-play/Settings/Language
**Depends on**: v1.1 complete
**Plans**: 1 plan

Plans:
- [x] 12-01: Header navigation bar with icon buttons, puzzle info display, clickable keyboard

**Details:**
- Header restructured with title on left, navigation icons on right
- Puzzle info (NYT vs Open mode + puzzle number) moved above grid
- On-screen keyboard made clickable for mobile users
- Reset button moved below keyboard for better visual hierarchy

### Phase 13: Settings Menu

**Goal**: Consolidate settings into modal: Light/dark mode, Hard mode, NYT mode toggle, Open mode toggle, Word list visibility, Hints toggle
**Depends on**: Phase 12
**Plans**: 1 plan

### Phase 14: Word Sync NYT

**Plans**: 1 plan

Plans:

### Phase 17: Hints System

**Goal**: 6 progressive hints per word, hidden by default, revealed sequentially via visibility icon click, final hint reveals answer
**Depends on**: Phase 16
**Plans**: 1 plan

## Milestone Summary

**Key Decisions:**
- Header position at top with left-aligned title, right-aligned icons
- Settings stored in single localStorage key
- JSON word data format for independent updates
- Open Mode synthetic puzzle uses game: -1 to distinguish from real puzzles
- Progressive hints: sequential unlocking enforced
- Portuguese word lists from IME-USP dictionary (CC BY license)
- Accent popup positioned above key (not below)

**Issues Resolved:**
- Keyboard not clickable on mobile (fixed in Phase 12)
- Accent popup occluded by finger (fixed in Phase 18-04)
- Key turning yellow during long-press accent interaction (fixed in Phase 18-04)

**Issues Deferred:**
- Hard mode guess validation (tracked in STATE.md Pending Todos)

**Technical Debt Incurred:**
- None significant
